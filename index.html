Kaza First Web Learning

1. HTML

1. Belajar tentang head dan body

Disini aku mempelajari bahwa ada yang namanya head sebagai kepala dan info tidak terlihat untuk web, tag yang merupakan fungsi perintah dan cara mengaplikasikannya di index pada HTML. dan aku belajar bahwa setiap halaman web biasanya punya judul bab dan subbab yang diaplikasikan pada fungsi:

  1. h1 = Judul
  2. h2 = Bab atau Bagian besar
  3. h3 = Subbab berupa keterangan bab atau bagian besar

2. Belajar tentang penyusunan kata, kalimat dan paragraf

Disini aku mempelajari hal seperti tag berupa br dan p, teks penting, menambah dan mengatur scale gambar, list, tabel dan struktur web.

  1. Penggunaan br dan p
    2. Aku belajar bahwa br dan p adalah tag dengan fungsi berbeda, p adalah paragraf dimana web akan otomatis memberikan jarak setiap paragraf. dan br adalah tag dengan fungsi untuk membuat baris baru.
    contoh:
    ini baris pertama
    ini baris kedua
  2. Teks penting
    3. Aku belajar tentang bagaimana membuat penekanan dan penjelasan terhadap teks atau kata yang diinginkan dengan melakukan bold dan italic pada bagian yang diinginkan dengan menggunakan tag strong sebagai bold dan tag em sebagai italic.
    contoh:
    Ini Bold memakai tag strong
    Ini italic memakai tag em
  3. Menambah dan mengatur scale gambar
    4. Aku belajar bahwa kita bisa menambahkan gambar dengan tag img, pada tag img akan ditambahkan src untuk sumber gambar yang diinginkan, width dan height untuk mengatur scale gambar, dan alt untuk catatan bila gambar tidak muncul. Kita bisa menambahkan gambar dari sumber folder yang sama dengan folder yang dipakai selama gambar tidak rusak dan hilang atau terhapus.
    contoh:

    Ini adalah contoh gambar yang menggunakan scale 300
  4. List
    5. Aku belajar bahwa kita bisa melakukan list dengan simpel dan mudah menggunakan tag ul sebagai list titik dan tag ol sebagai list angka.
    contoh:
    • List titik pertama
    • List titik kedua

    1. List angka pertama
    2. List angka kedua
  5. Tabel
    6. Aku belajar bahwa kita bisa membuat tabel simpel dengan tag tr, th dan td sebagai contoh:
    No. Tag Fungsi
    1. tr baris
    2. th judul kolom
    3. td isi
  6. Struktur web
    7. Aku belajar bahwa kita bisa mengatur struktur web dengan menggunakan tag header, main, dan footer untuk membuat web menjadi lebih terstruktur dan rapi sehingga browser bisa membaca web kita lebih baik. Untuk contoh penerapan tidak terlihat dan telah diaplikasikan pada web ini.
